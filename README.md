# 📦 Hash Extensível em C

Implementação de uma **Tabela Hash Extensível** em linguagem C, utilizando diretório dinâmico e buckets com profundidade variável.

---

## 📚 Sobre o Projeto

Este projeto implementa uma estrutura de dados baseada em **Hash Extensível**, muito utilizada em sistemas de banco de dados para indexação eficiente em disco.

### 🔥 Características

- Crescimento dinâmico
- Baixa colisão
- Divisão automática de buckets (split)
- Estrutura escalável
- Encapsulamento com tipos opacos
- Testes unitários com Unity

---

## 🧠 Conceitos Utilizados

- Hash baseado em bits menos significativos
- Diretório com profundidade global
- Buckets com profundidade local
- Rehash após divisão
- Compartilhamento de buckets
- Gerenciamento de memória em C
